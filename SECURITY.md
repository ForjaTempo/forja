# Security Policy

## Supported Versions
The `main` branch is the only supported version. forja.fun runs from main.

## Reporting a Vulnerability
Please do NOT open a public GitHub issue for security issues.
Email: security@forja.fun
Response SLA: within 72 hours.
Full disclosure policy: https://forja.fun/security

## Scope
See https://forja.fun/security — in-scope contracts, severity tiers, safe-harbor terms.
